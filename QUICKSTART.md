# 🚀 Quick Start Guide

## Start the Application in 3 Simple Steps

### Option 1: Using the Startup Script (Recommended)

```bash
# Make the script executable (first time only)
chmod +x start.sh

# Run the startup script
./start.sh
```

### Option 2: Manual Start

```bash
# Install dependencies
pip install -r requirements.txt

# Run the application
python3 app.py
```

## Access the Application

1. Open your browser
2. Navigate to: **http://localhost:5000**
3. Login with:
   - Username: `admin`
   - Password: `admin123`

## Test the System

1. Click **ARM SYSTEM** button
2. Click **SIMULATE INTRUSION** button
3. See the red alert banner appear
4. Check the console for "Emergency services notified" message
5. Go to **Event History** to view logged events

## Stop the Application

Press `Ctrl+C` in the terminal where the app is running.
